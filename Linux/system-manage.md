## 后台运行程序

### 后台运行的两种方式
- “命令 &”
  - 执行的命令不能与前台有交互

- 命令执行过程中，按住 Ctrl+Z 快捷键
  - 处于暂停状态

### 查看当前终端放入后台的 jobs
```
jobs -l

[1]+  2957 Stopped (signal)        top
```
[1] 代表工作号 <br />
"+" 号代表最近一个放入后台的工作 <br />
"-" 号代表倒数第二个放入后台的工作

### 把后台程序恢复到前台执行
```
fg %工作号
```

### 把后台程序恢复到后台执行
```
bg %工作号
```
### 后台程序脱离终端运行
```
nohup [命令] &
```
